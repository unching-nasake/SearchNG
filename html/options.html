NGW4B - Options

NG List

Options can be added at the end of a line in the following format.

[option1(, option2, ...)]

Option list:

  • notitle - Exclude page title from check
  • nosite - Exclude site name from check
  • nodesc - Exclude page description from check
  • regex - Enable Regular Expressions
